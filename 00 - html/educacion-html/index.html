Conceptos Básicos de HTML

Ejemplos educativos basados en W3Schools

¿Qué es HTML?

HTML es el lenguaje estándar para crear páginas web. Significa HyperText Markup Language.

Estructura Básica de una Página HTML

<!DOCTYPE html>
<html>
  <head>
    <title>Título de la página</title>
  </head>
  <body>
    <h1>Mi primer encabezado</h1>
    <p>Mi primer párrafo.</p>
  </body>
</html>

Encabezados HTML

<h1>Este es un encabezado grande</h1>
<h2>Este es un encabezado mediano</h2>
<h3>Este es un encabezado pequeño</h3>

Párrafos HTML

<p>Este es un párrafo.</p>
<p>Este es otro párrafo.</p>

Enlaces HTML

<a href="https://www.w3schools.com">Visita W3Schools</a>

Imágenes HTML

<img src="https://www.w3schools.com/html/img_girl.jpg" alt="Niña" width="200">

Botones HTML

<button>Haz clic aquí</button>

Elementos HTML

Un elemento HTML normalmente consta de una etiqueta de apertura, contenido y una etiqueta de cierre. Algunos elementos no tienen contenido ni etiqueta de cierre (elementos vacíos).

<p>Este es un párrafo</p>
<br>
<img src="https://www.w3schools.com/html/img_girl.jpg" alt="Ejemplo Imagen">
  • Elemento con contenido: <p>Hola</p>
  • Elemento vacío: <br> o <img>

Los elementos HTML pueden estar anidados unos dentro de otros. Por ejemplo:

<div>
  <p>Un párrafo dentro de un div.</p>
</div>

Ejemplo: Encabezados HTML

Los encabezados HTML van de <h1> (más importante) a <h6> (menos importante):

<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>
<h3>Encabezado 3</h3>
<h4>Encabezado 4</h4>
<h5>Encabezado 5</h5>
<h6>Encabezado 6</h6>
